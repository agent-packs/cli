Why Agent Packs

Everything Homebrew does,
for AI Skills & Plugins

One CLI to install, update, audit, and rollback skills & plugins across every AI coding agent you use.

🤖
One Tool, All Agents
Manage skills and plugins across Claude Code, Codex, Cursor, Gemini CLI, and more from a single CLI. Install once, track everywhere.
agent-packs install backend-engineer --agent claude
📦
Tap System
Add third-party skill registries from any GitHub repository, just like Homebrew taps.
agent-packs tap myorg/skills
🛡️
Trust Tiers
Skills and plugins are rated core → community → tap → unverified. Set a minimum trust floor before installing.
agent-packs install backend-engineer --min-trust community
🔍
Drift Detection
See exactly what changed in an installed pack since you last updated — before pulling new changes.
agent-packs diff backend-engineer
⏪
Rollback
Every install writes a lockfile. Roll back to the previous version of any pack instantly.
agent-packs rollback backend-engineer
🔗
Dependency Resolution
Packs declare required skills and conflict guards. Auto-expanded before install — no manual juggling.
agent-packs show rust-engineer

How it works

Packs: one bundle,
every capability

A Pack is a named collection of Skills and Plugins tuned for a role — backend engineer, ML engineer, SRE. One install command resolves every skill and plugin to the right directory for every agent you use. Packs can even compose other packs, so a full-stack pack just references the backend and frontend packs — no duplication.

Getting Started

Up and running in minutes

1

Install the CLI

Install via curl, Homebrew, Docker, or Go.

# curl (macOS / Linux)
curl -fsSL https://raw.githubusercontent.com/agent-packs/cli/main/install.sh | sh

# Homebrew
brew install agent-packs/tap/agent-packs

# Docker
docker run --rm -it ghcr.io/agent-packs/cli:latest
2

Search the registry

Find packs by role, language, or tag.

agent-packs search backend
# → backend-engineer, rust-engineer …
3

Install into your agent

Pick a tool and install. With --mode copy, skills land in the right directory automatically (the default reference mode records sources without copying).

agent-packs install backend-engineer --agent claude --mode copy
agent-packs install sre-engineer --agent cursor --mode copy
# Installed 7 skills → ~/.claude/skills/
4

Keep up to date

Update all installed packs, or just one. Diff before you upgrade.

agent-packs outdated
agent-packs diff backend-engineer
agent-packs upgrade backend-engineer
